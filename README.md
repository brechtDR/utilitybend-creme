# Utilitybend Creme

A warm creme light theme for VS Code, Cursor, VSCodium, and any Open VSX-compatible editor. The light sibling of [Utilitybend Neon](https://github.com/brechtDR/utilitybend-neon), built from the Prism light palette used on [utilitybend.com](https://utilitybend.com), tuned for long sessions of writing HTML, CSS/SCSS, JS/TS, JSON, YAML, and Markdown.

## Palette

| Role | Hex | Swatch |
| --- | --- | --- |
| Background | `#f6f3ec` | warm creme |
| Foreground | `#222222` | near black |
| Comments | `#616161` | neutral grey |
| Punctuation | `#bc0376` | magenta |
| Keywords / Functions / Constants | `#7b5296` | dusty purple |
| Tags / Templates / Headings | `#eb007d` | hot pink |
| Strings / Pseudo-classes / Attribute values | `#5d8a1c` | olive green |
| Attribute names / Types / Classes / Builtins | `#a24c05` | dark amber |
| Operators / Properties / Custom props | `#05618f` | deep blue |
| Numbers / Units / At-rules / Decorators | `#f93d04` | red-orange |
| Variables / Errors / Deleted | `#c0392b` | brick red |
| CSS selectors | `#ff00d6` | neon magenta |
| Regex / class names / hex values | `#6d3a00` | deep brown |

## Install

### VS Code / VS Code Insiders
- Extensions panel -> search **"Utilitybend Creme"** -> Install
- Or CLI: `code --install-extension utilitybend.utilitybend-creme`
- Or direct link: <https://marketplace.visualstudio.com/items?itemName=utilitybend.utilitybend-creme>

### Cursor / VSCodium / Gitpod / code-server
- Extensions panel -> search **"Utilitybend Creme"** (these editors query Open VSX)
- Or CLI: `cursor --install-extension utilitybend.utilitybend-creme`
- Or direct link: <https://open-vsx.org/extension/utilitybend/utilitybend-creme>

## Activate

`Cmd/Ctrl + K` then `Cmd/Ctrl + T` -> pick **Utilitybend Creme**.

## Designed for

- HTML / JSX / TSX
- CSS / SCSS / Sass (including modern custom properties, pseudo-classes, at-rules)
- JavaScript / TypeScript (semantic highlighting enabled)
- JSON / YAML
- Markdown
